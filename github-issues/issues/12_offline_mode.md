# Offline/local practice mode

As a user, I want a local/offline mode for short practice tests so I can practice without internet.

Acceptance criteria

- Given no network, when I open the app, then local practice texts work and results are stored locally (sync later if desired).

Definition of done

- Offline practice mode implemented with local storage fallback.
