# Difficulty selection (beginner/intermediate/advanced)

As a user, I want to choose difficulty (beginner/intermediate/advanced/custom) so the text matches my ability.

Acceptance criteria

- Given difficulty options, when I select one and start a test, then the provided text difficulty matches that level.
- The app should explain what each level means.

Definition of done

- Difficulty selection implemented, content mapped to levels, and basic QA verifying alignment.

Priority: High
Estimate: 3 pts
Labels: user-story, priority:high
