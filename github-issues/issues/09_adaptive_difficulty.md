# Adaptive difficulty suggestions

As a returning user, I want adaptive difficulty that suggests tests based on my past performance so I can progressively improve.

Acceptance criteria

- Given my past results, when I open the test selector, then the app suggests an appropriate level (example heuristic: WPM > 50 and >95% accuracy => suggest intermediate/advanced).

Definition of done

- Simple heuristics implemented and suggestions shown.

Priority: Medium
Estimate: 5 pts
