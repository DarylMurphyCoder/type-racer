# Per-word/per-character error highlighting

As a user, I want to see a per-word/per-character error highlight during and after the test so I can learn from mistakes.

Acceptance criteria

- Given I'm typing, when I mistype, then the character or word is highlighted.
- Given results shown, when I view them, then each error shows expected vs typed text.

Definition of done

- Visual highlights and result detail implemented; verified on desktop/mobile.

Priority: Medium
Estimate: 3 pts
Labels: user-story, priority:medium
