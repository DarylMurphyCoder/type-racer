# Clear instructions / How to take the test

As a new user, I want clear instructions on how to take the test so I know what to expect.

Acceptance criteria

- Given I land on test page, when I click "How it works" or a first-run tooltip, then I see concise guidance: what the timer does, how WPM is calculated, how errors are handled, and keyboard focus behavior.

Definition of done

- Info panel or modal with short steps and a dismiss option; copy reviewed for clarity.

Priority: High
Estimate: 2 pts
