# Timed typing test: measure current typing speed

As a user, I want to take a timed typing test so I can measure my current typing speed.

Acceptance criteria

- Given I’m on the test page, when I start the test, then a timer begins and typed keystrokes are recorded.
- Given the test ends (timer reaches zero or I finish the text), when results are calculated, then I see my WPM, accuracy percentage, error breakdown (mistyped words/characters), and raw typed characters.

Definition of done

- Timer, input capture, WPM & accuracy calculation, and results UI implemented and manually validated.

Priority: High
Estimate: 5 pts
