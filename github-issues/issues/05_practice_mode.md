# Practice test mode

As a user, I want a short demonstration/run-through so I can try a practice test before measuring.

Acceptance criteria

- Given a "Practice" option, when selected, then I can take a short sample test that does not save to main progress but demonstrates features (timing, highlighting).

Definition of done

- Practice mode available and excluded from progress metrics.

Priority: Medium
Estimate: 2 pts
