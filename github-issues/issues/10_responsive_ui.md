# Responsive UI for multiple devices

As a user, I want the test UI to be responsive so I can use the app on desktop, tablet, and phone.

Acceptance criteria

- Given different viewport widths, when I load the test, then layout adapts (text wraps, controls are reachable, font sizes appropriate).

Definition of done

- Responsive CSS with breakpoints and manual checks on at least 3 screen sizes.

Priority: High
Estimate: 2 pts
Labels: user-story, priority:high
