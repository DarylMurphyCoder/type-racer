# Export/share results

As a user, I want to export or share my results (image or CSV) so I can show progress or analyze offline.

Acceptance criteria

- Given results, when I click Export/Share, then I can download CSV with test date, WPM, accuracy, text id; or generate a shareable image summary.

Definition of done

- Export CSV and share image implemented (or share sheet integration).

Priority: Low
Estimate: 3 pts
Labels: user-story, priority:low
