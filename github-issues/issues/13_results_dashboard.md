# Results dashboard and progress trends

As a user, I want to see a results dashboard showing trends (WPM over time, accuracy trends) so I can track improvement.

Acceptance criteria

- Given multiple saved test results, when I open the dashboard, then graphs show WPM and accuracy across time and highlight personal bests.

Definition of done

- Basic trend chart + personal best and total test count implemented.

Priority: Medium
Estimate: 5 pts
