# Auto-focus input for immediate typing

As a user, I want the test input to auto-focus and accept keyboard input so I can start typing immediately.

Acceptance criteria

- Given I open the test page, when I press Start, then the input is focused and keyboard input is captured without additional clicks.
- Support for mobile virtual keyboards (tap to focus).

Definition of done

- Auto-focus behavior implemented and verified on desktop/mobile.

Priority: High
Estimate: 1 pt
Labels: user-story, priority:high
